# Templates

> 规范模板

## 特性规范模板

```markdown
# Feature: <功能名称>

## 概述
<功能简要描述>

## 功能需求
- [ ] <需求 1>
- [ ] <需求 2>

## 非功能需求
- 性能：<要求>
- 可用性：<要求>
- 安全：<要求>

## 验收标准
1. <标准 1>
2. <标准 2>
```

## 变更规范模板

```markdown
# Change: <变更描述>

## 变更原因
<为什么需要变更>

## 变更内容
- <变更点 1>
- <变更点 2>

## 影响范围
<影响的模块/功能>

## 回滚计划
<如何回滚>
```
